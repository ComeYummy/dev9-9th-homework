写真共有サイト
====

## Description
写真共有サイトです。
titleとPASSを入れると写真の一覧が表示されます（ログイン認証）。
マルチデバイス対応しているはずです。
写真ごとのタグ編集も可能です。
※サイドメニューからタグ一覧表示まで実装完了。「タグを押して写真一覧画面表示」は未実装。

## Demo
index.htmlが初期画面です。
好きなtitleを入力してください。
ユーザ側：
"title:abc", "pass:abc"と入力すると既に登録されているデータがみれます。

manage.phpがデータベースに管理された一覧画面です。
管理者側：
"title:manager", "pass:manager"と入力すると既に登録されているデータがみれます。
